# Dokumentasi CORELASI

Folder ini menyimpan dokumentasi project yang dibutuhkan untuk perencanaan, arsitektur, QA, UI/UX handoff, deployment, dan context product.

## Subfolder

- `architecture/`: rencana arsitektur dan pembagian modul.
- `context/`: SRS dan catatan product context.
- `qa/`: template test case, bug report, regression, dan sign-off.
- `uiux/`: link Figma, user flow, design handoff, dan review visual.
