# Project Context

## Product

CORELASI adalah sistem administrasi akademik sekolah untuk mendukung pengelolaan data akademik, jadwal, presensi, pembelajaran, jurnal mengajar, dan laporan.

## Role Aplikasi

- Admin: mengelola data user, akademik, jadwal, presensi, jurnal, dan laporan.
- Guru: mengelola presensi kelas, materi, tugas, jurnal, nilai, dan laporan kelas.
- Siswa: melihat jadwal, presensi, materi, tugas, pengumpulan, dan nilai.

## Kolaborasi

Pengerjaan mengikuti Jira Scrum. Setiap anggota menggunakan branch sesuai task dan hanya mengubah area file yang menjadi ownership role masing-masing.
